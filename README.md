## 🚀 使用方法
1. **环境要求**  
   - Windows 10/11  
   - [AutoHotkey v2](https://www.autohotkey.com/download/ahk-v2.exe)

   - 下载 `.ahk` 文件  
   - 双击运行，或右键 → “使用 AHK v2 运行”  
   - 按需在 GUI 中设置

   - 打开脚本直接改热键/功能即可 ✍️
